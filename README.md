# AIFoundation

## 课程背景

聚焦 **AI Foundation**，大模型系统。大模型是基于 AI 集群的全栈软硬件性能优化，通过最小的每一块 AI 芯片组成的 AI 集群，编译器使能到上层的 AI 框架，训练过程需要分布式并行、集群通信等算法支持，而且在大模型领域最近持续演进如智能体等新技术。

![大模型系统全栈](images/aifoundation.png)
